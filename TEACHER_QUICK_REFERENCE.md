# 📇 教師功能快速參考卡片

## 🎯 三個新功能一覽

```
┌─────────────────────────────────────────────────────────────┐
│              教師功能 - 快速參考卡片                        │
├─────────────────────────────────────────────────────────────┤
│                                                              │
│  1️⃣  新增課程 (Add Course)                                 │
│  ├─ 目的: 建立新課程                                        │
│  ├─ 訪問: http://127.0.0.1:8000/teacher/add_course/       │
│  ├─ 時間: ~1 分鐘                                          │
│  ├─ 操作: 填寫 2 個必填欄 → 點擊「新增課程」             │
│  └─ 結果: 課程自動指派給你 ✅                             │
│                                                              │
│  2️⃣  查詢修課學生 (View Students)                         │
│  ├─ 目的: 查看誰修了你的課                                 │
│  ├─ 訪問: http://127.0.0.1:8000/teacher/student_list/    │
│  ├─ 時間: ~30 秒                                          │
│  ├─ 操作: (選擇課程篩選) → 看學生表格                      │
│  └─ 結果: 顯示期中、期末、平均分數 ✅                     │
│                                                              │
│  3️⃣  輸入成績 (Grade Entry)                               │
│  ├─ 目的: 給學生打期中、期末分                             │
│  ├─ 訪問: http://127.0.0.1:8000/teacher/grade_entry/<id>│
│  ├─ 時間: ~2 分鐘                                         │
│  ├─ 操作: 輸入分數 → 點擊「保存所有成績」              │
│  └─ 結果: 成績實時計算平均值 ✅                          │
│                                                              │
└─────────────────────────────────────────────────────────────┘
```

## 📱 訪問方式對比

### 方式一: 從教師儀表板 (推薦)
```
1. 登入教師帳號
2. 進入「👨‍🏫 教師」儀表板
3. 點擊快速操作按鈕
   ├─ ➕ 新增課程
   ├─ 👥 查詢修課學生
   └─ 📊 輸入學生成績
```

### 方式二: 直接 URL 訪問
```
新增課程:    http://127.0.0.1:8000/teacher/add_course/
查詢學生:    http://127.0.0.1:8000/teacher/student_list/
成績輸入:    http://127.0.0.1:8000/teacher/grade_entry/<course_id>/
```

### 方式三: 從學生列表進入
```
1. 進入「查詢修課學生」
2. 篩選特定課程
3. 點擊「編輯此課程成績」
4. 進入成績輸入頁面
```

## 🆘 故障排除速查

### 問題: 無法訪問頁面

```
✓ 伺服器運行?    → 檢查 http://127.0.0.1:8000/
✓ 已登入?        → 檢查右上角顯示用戶名
✓ 是教師嗎?      → 檢查 Django Admin 中的 User Profiles
✓ URL 正確?      → 複製並重新粘貼
```

### 問題: 課程代碼重複

```
✓ 解決: 使用不同的代碼
  - CS101  → CS101_2025
  - CS101  → CS101_A
  - CS101  → CS101_B
```

### 問題: 成績無法保存

```
✓ 分數範圍?  → 必須 0-100
✓ 是數字?    → 不能有字母或符號
✓ 有權限?    → 必須是課程授課教師
✓ 格式?      → 支持小數 (如 85.5)
```

### 問題: 平均分未計算

```
✓ 瀏覽器支持 JavaScript?
✓ 嘗試重新載入頁面 (F5)
✓ 嘗試使用 Chrome 或 Firefox
✓ 檢查瀏覽器控制台是否有錯誤
```

## 🎯 使用提示

### 最佳實踐

```
✅ 課程代碼命名
  CS101, CS102, CS201, CS301  (清晰的編號系統)
  不要: 1, 2, a, b              (太簡略)

✅ 成績輸入
  一次性輸入所有學生            (提高效率)
  檢查無誤後再保存               (避免錯誤)

✅ 學生名單管理
  定期查看新註冊學生             (及時掌握)
  及時輸入成績                   (學生可查看)
```

### 常見操作

```
操作 1: 快速建立課程
  1. 點擊「➕ 新增課程」
  2. 輸入課程名稱和代碼
  3. 點擊「新增課程」 ✓

操作 2: 快速評分
  1. 點擊「👥 查詢修課學生」
  2. 篩選課程
  3. 點擊「編輯此課程成績」
  4. 輸入分數
  5. 點擊「保存所有成績」 ✓

操作 3: 查看成績
  1. 進入「查詢修課學生」
  2. 篩選課程
  3. 看表格中的期中、期末、平均值 ✓
```
